Тик-так-бум

В этой игре несколько игроков должны по очереди отвечать на вопросы, чтобы обезвредить бомбу.

Правила:

  • У каждого игрока время бомбы свое.
  • За правильный ответ игроку добавляется пять секунд, за неправильный — отнимается.
  • У каждого игрока по 3 жизни. За каждый неправильный ответ отбирается одна жизнь.
  • Игрок, допустивший истечение времени таймера, или потерявший все жизни — выбывает.
  • Игра может сделать случайный вопрос "золотым". Игрок, давший на него верный ответ побеждает сразу.
  • Игра может сделать случайный вопрос "опасным". При неправильном ответе на него, бомба взрывается для всех.
  • Если у игры заканчиваются предустановленные вопросы, она начинает генерировать случайные задачи на сложение. В этом режиме игрок ответивший неправильно, теряет все оставшиеся жизни.
  • Игрок, оставшийся последним, должен ответить на один вопрос для победы.
